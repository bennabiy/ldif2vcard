The following were goals for ldif2vard:

  1.  Create iPod compatible vCards from LDAP
  
  2.  Capable of including contact photos
  
  3.  Scriptable - perl
  
  4.  Lightweight - it is only 8K!
  
  5.  Portable - perl will run on nearly any system, especially *nix flavors
